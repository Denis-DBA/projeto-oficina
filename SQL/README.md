
Esta pasta contém todos os scripts SQL utilizados no desenvolvimento do projeto **Projeto Oficina**.

## Conteúdo

- **Projeto_Oficina.sql** – Script completo do banco de dados.
- Criação das tabelas.
- Constraints (Primary Key e Foreign Key).
- Inserção de dados.
- Views.
- Procedures.
- Functions.
- Triggers.

## Como utilizar

1. Abra o MySQL Workbench.
2. Crie um banco de dados.
3. Execute o arquivo `Projeto_Oficina.sql`.
4. O banco será criado com toda a estrutura e os dados de exemplo.

## Tecnologias

- MySQL
- SQL
- MySQL Workbench
